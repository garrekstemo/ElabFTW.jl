Team

Team-scoped tag management and category listing.

Tags

ElabFTW.list_team_tags — Function
list_team_tags() -> Vector{Dict}

List all tags in the team registry.

Example

tags = list_team_tags()
for t in tags
    println(t["id"], ": ", t["tag"])
end
ElabFTW.rename_team_tag — Function
rename_team_tag(tag_id::Int, new_name::String)

Rename a tag in the team registry. Admin-only operation.

Example

rename_team_tag(7, "ftir-analysis")
ElabFTW.delete_team_tag — Function
delete_team_tag(tag_id::Int)

Delete a tag from the team registry. Removes the tag from all entity references. Admin-only operation.

Example

delete_team_tag(7)
